- **해당 주차에 이슈 문제 풀어오기**

---

## 📁 저장 구조(폴더 컨벤션)

> **형식:** `[개인폴더]/.../[주차]/[파일명]/[레벨]`

### 파일명 규칙
- **프로그래머스:** `{문제이름}.{확장자}`  
  - 예) `문제이름.java`
---

## 🗓️ 운영 방식

### 1) 문제 풀이 (주중)
- 각자 **문제 풀이 후 PR 제출**  
- 풀이마다 주석을 간단히 작성

### 2) 리뷰 (수요일)
- **문제 풀이 리뷰 진행**
- 서로의 풀이를 보고 대안 제시/질문 남기기
