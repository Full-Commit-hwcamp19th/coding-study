## 🗓️ 운영 방식

1) **문제 선정**  
   - 매주 선정된 프로그래머스 문제 GitHub **이슈에 확인 댓글 달기**

2) **문제 풀이**  
   - 각자 이슈에 올라온 문제 풀이 후, 풀이 과정을 개인 폴더에 정리  
   - 풀이마다 주석 작성  

3) **리뷰 (수요일)**  
   - 모여서 각자 풀이 리뷰  
   - **서로의 접근 방식 비교**, 대안 제시, 질문 및 피드백  
<br>

## 📁 저장 구조 (폴더 컨벤션)

> **형식:** `[개인폴더]/.../week00/lv1/[문제이름].[확장자]`
